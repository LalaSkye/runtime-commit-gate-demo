# Public Disclosure Boundary

This repository is a public inspection surface, not full architecture disclosure.

It shows only what is needed to inspect the specific public claim attached to this repo: the bounded evidence object, the inspection path, and the claim limit.

It must not be treated as a map of the wider system, a disclosure of protected architecture, or evidence of production deployment unless explicitly stated.

Public inspection standard:

```text
claim -> evidence object -> inspection path -> claim limit
```

A local proof object proves only the local claim attached to it.
